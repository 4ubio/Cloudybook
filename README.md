## Run Locally

Clone the project

```bash
  git clone https://github.com/4ubio/Cloudybook.git
```

Go to the project directory

```bash
  cd Cloudybook/
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```

```bash
  yarn host
```
